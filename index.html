Airplane Configurator

The project helps sales and marketing team of a leading aircraft manufacturer in USA to set up and configure airplane interiors providing various options such as quantity of Seats (Business, First and Economy), type of Lavatories, Galleys, Video Controllers etc., while taking orders from airline customers. This application helps to check cabin layouts to ensure compliance with physical constraints, certification requirements and corporate design policies. It gives a provision to create 2D and 3D product configurations with interactive, drag-and-drop technology and extending it to virtual reality and 3D walkthroughs of Cabin Layouts. The project helped in improving digital customer experience by enhancing the visual representation of commercial airplane interiors configurations from traditional paper-based drawings and PDF catalogs.

Galley ICM

Galley inserts configuration management is an angular based web application used to configure and manage parts for each effectivity. The application connects to various upstream and downstream systems to pull the data on daily basis. Configuring parts for each option, parts management, supplier management, tracking schedule, exporting determinations, identifying the risk items and creating action items should be the key functionalities of the system. All the configuration for each effectivity is currently done manually. It is very time consuming to identify the qualifying documents and identifying the certification information for each part. ICM configuration manager helps to address this issue.
