Quality Dashboard - An Analytics Application

Quality dashboard Web Application is a data analytics platform developed to monitor quality notes and concessions which in turn provides the Quality leaders/team with the relevant and timely information. Consolidated data from SAP systems, E-Concession and Net Inspect Systems will be used as data sources to analyze the allocation and concession process. Application performs various analytics and provides insights of the data to the end user.


Note : In order to maintain confidentiality, client name has not been disclosed.

Environment

  • Silverlight
  • WCF RIA
  • Data Analytics
  • Design / Architect
  • SSIS / SQL Server
  • SSRS
  • Security layer

Year

2012 – 2013

Silverlight based analytical application

My role in the project is to perform Data Manipulation, Business Intelligence and Dashboard Techniques deriving insights from data. Application was implemented using Silverlight 3 toolkit for data visualization options (Charts and Graphs). For complex charts I used RadChart for Silverlight, a third-party tool to transform various business scenarios into interactive, rich, animated charts that enable the end users to analyze complex data.

Architecture

Silverlight was new those days with a sophisticated architecture. I was one of the key people in the project to architect the application for different components. For report generation we were using SQL Server Reporting Services (SSRS). Silverlight has mandate to use services using Windows Communication Foundation (WCF RIA). I also participated in SSIS ETL pipelines with ERP systems and other upstream systems for data.
