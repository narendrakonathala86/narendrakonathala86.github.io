Design Checker Tool

Design checker is sophisticated rich UI based application suite for Design engineers to perform design checks integrating CATIA, PDM and ENOVIA. The tool performs 95 design checks which comprises of installation drawings checks, machined parts, Sheet metal parts and tubing related. The tool is highly modular with all the configurations and rules for performing the check in built making it highly scalable and intuitive user experience


Note : In order to maintain confidentiality, client name has not been disclosed.

Year

2015

93 Design checks

The tool helps the designers to check 2D drawings and 3D models are as per design specifications, check accuracy of the BOM, checks consistency between 2D drawings and 3D models, consistency between 2D drawings and BOM within PDMlink SSCI and consistency between 3D models and BOM within PDMlink SSCI Application.

My role in the project is design the prototypes and validate with designers at onsite, create WPF styles and resources following customer theme, gather requirements from various teams, perform workshops and training sessions to make users adopt to the system.

Customizations / Configurations

I was also resposible to perform a lot of user interviews, user feedback and designer inputs across teams, so that we can address all the key issues the end user face. This made us develop the tool to be fully customizable, with both user specific and admin specific customizations. A designer friendly and easy to use tool helped us to get a good appreciation with this project.
