Galley Inserts Configuration Management

ICM Galley Inserts is an angular based web application used to configure and manage parts for each effectivity. The application should connect to various upstream and downstream systems to pull the data on daily basis. Configuring parts for each option, parts management, supplier management, tracking schedule, exporting determinations, identifying the risk items and creating action items should be the key functionalities of the system. All the configuration for each effectivity is currently done manually. It is very time consuming to identify the qualifying documents and identifying the certification information for each part. ICM configuration manager helps to address this issue.

Angular JS Web Application

Galley insert team comprises of 200 member engineers, each having their own way of managing and configuring inserts. Incorporating the functionality was not challenging task. However, gathering feedback from multiple users making them adopt to use the system was more challenging. It took us multiple feedback implementation; training sessions and user interviews which made this application successful.

My role in the project is develop the front end using angular and backend as ASP.Net Web API. At offshore, I am also responsible for leading a team of 10 people working on this project. Performing code reviews, creating custom controls, training freshers, writing unit tests and deploying code to onsite were few of my roles in the project.

Data warehousing

The application uses Entity Framework 6.0 as data layer with code first approach. I incorporated unit of work and repository design pattern on top of EF6.0 DBcontext and used as our data layer. I was also responsible for creating SSIS jobs to perform ETL operations from different sources. A scheduler job was created to update the data periodically, making the system up to date.
